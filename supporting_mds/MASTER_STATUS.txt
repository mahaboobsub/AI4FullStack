SECTION 6 — KNOWN ISSUES & NOTES
================================================================================

1. ngrok authtoken required (free signup at ngrok.com)
   Without it: Telegram webhook cannot be set → bot won't receive messages

2. VITE_STAFF_TOKEN = "test-admin-token" is a placeholder
   Admin pages (analytics, traces) will return 401 until real token is set
   Get real token: Supabase → staff table → auth_token column

3. Neo4j may be empty — if blood bank/graph pages show no data,
   run POST /api/donors to create donors (they auto-build Neo4j edges)

4. Bolna.ai phone number required for voice calls
   Set up Plivo/Exotel number in Bolna Telephony settings

5. Frontend pnpm workspace install needed for production build
   Dev mode works with: node node_modules\vite\bin\vite.js

6. OCR service (Tesseract) requires system-level install
   For Windows: choco install tesseract OR use Docker
   For production: already in Dockerfile/Render build commands

7. DonorPortal shows "Hero" if no donor_id in localStorage
   Fix: log in via /donor/login first (enters donor ID D-1001 by default)
